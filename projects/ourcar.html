✅ 사용 기술 스택
✅ 서비스 상세 요소

1. 🔧기능 측면

  • 제조사별, 모델명별, 연식별 등 각종 필터 제공
  • 마이페이지에서 판매 내역, 구매 내역, 거래 내역 확인 기능
  • Google Maps를 연동하여 판매자가 제시한 거래 지역 확인
  • 연간 평균 주행거리로 주행거리 적절/부적절 판단

2. AI 모델 측면

  • 🚗차량 탐지 모델
    • Yolov8을 활용해 이미지에서 차량 객체가 탐지된 이미지만 매물 이미지로 업로드할 수 있습니다.
  • 🌩차량 파손 부위 탐지 모델
    • Yolov8을 fine-tuning하여 파손을 탐지할 수 있습니다.
  • 💸차량 적정 가격 예측 모델
    • RandomForest Regressor 모델로 25,000개의 중고차 데이터 학습
  • 💰구매자 예산 추천 모델
    • 구매자의 카드 데이터를 기반으로 유사한 소비 패턴을 지니는 다른 고객과 유사한 가격대의 차량 예산을 추천
  • 📱신용평가 예측 모델
    • LendingClub의 P2P 데이터 150개 컬럼 15만 건 학습.
    • Score Card 모델링, Logistic Regression 모델링
    • 모델이 예측한 신용 정보를 기반으로 가장 적합하고 금리가 저렴한 대출 상품 추천
    • 정확도 약 90%
